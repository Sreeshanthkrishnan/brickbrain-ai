# Repository Configuration Guide

## GitHub Settings

### 1. GitHub Actions Permissions
Ensure repository workflow permissions are set to:
- **Read and write permissions** (Settings -> Actions -> General -> Workflow permissions)

### 2. GitHub Pages Setup
- Go to `Settings` -> `Pages`
- Under **Build and deployment**:
  - Source: **Deploy from a branch** (Branch: `gh-pages` / `/ (root)`) or **GitHub Actions**
- Live published report URL:
  `https://Sreeshanthkrishnan.github.io/brickbrain-ai/reports/latest/execution-report.html`
